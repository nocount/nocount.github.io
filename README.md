# tensorwoe.com

Personal website for Wilson, a Data Engineer.

Built with vanilla HTML and CSS, hosted on GitHub Pages.

## Features

- Dark theme with Inter font
- Animated avatar with glow effect
- Fade-in page transitions
- Responsive design

## Pages

- **Home** - Landing page with avatar and tagline
- **About** - Bio, background, and skills
- **Projects** - Portfolio of work
- **Links** - Social media and other links
- **Contact** - Contact information
